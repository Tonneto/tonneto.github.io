HTML Tables
Tag / Attribute Description
<table> Defines a table.
<tr> Defines a row in a table.
<th> Defines a header cell in a table.
<td> Defines a standard cell in a table.
<caption> Defines a table caption.
<thead> Groups the header content in a table.
<tbody> Groups the body content in a table.
<tfoot> Groups the footer content in a table.
colspan Defines how many columns a td element should span.
rowspan Defines how many rows a td element should span.
CSS Properties and Descriptions
Property Description
color Defines the text color of an element.
background-color Defines the background color of an element.
font-family Defines the font family to be used for an element's text.
font-size Defines the text font size.
font-weight Defines the thickness (weight) of the font for the text, such as bold for bold text.
text-align Aligns the text of an element. Common values include left, right, center, and justify.
margin Defines the outer margin around an element. Can specify individual margins with margin-top, margin-right, margin-bottom, and margin-left.
padding Defines the internal spacing between the content of an element and its border. Similar to margin, individual paddings can be specified.
border Defines the border around an element. Can be divided into border-width, border-style, and border-color.
width Defines the width of an element.
height Defines the height of an element.
display Defines how an element is displayed on the page. Common values include block, inline, inline-block, and none.
position Defines how an element is positioned on the page. Common values include static, relative, absolute, and fixed.
overflow Controls what happens with content that overflows an element's box. Common values are visible, hidden, scroll, and auto.
background-image Defines a background image for an element.
opacity Defines the opacity of an element, where 0 is completely transparent and 1 is completely opaque.
z-index Controls the stacking order of positioned elements. Elements with a higher z-index are stacked above those with a lower z-index.
